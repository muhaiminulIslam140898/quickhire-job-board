# QuickHire – Job Board Application

QuickHire is a simple full-stack job board web application where users can browse jobs, view job details, and apply for positions. An admin panel allows job posting and management.

## Tech Stack

Frontend
- React
- TypeScript
- Tailwind CSS

Backend
- Node.js
- Express.js
- MongoDB

## Features

User Features
- View job listings
- Search jobs
- View job details
- Apply for jobs

Admin Features
- Create job listings
- Delete jobs
- View applications

## Project Structure

quickhire/
frontend/ (React + TypeScript)
backend/ (Node + Express API)
